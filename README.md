# Raising Taxes

*A Discworld novel written by Claude Fable 5, with the utmost respect to Sir Terry Pratchett.*

Moist von Lipwig has run out of things to save — so Lord Vetinari hands him
the one institution no one has ever survived reforming: the taxes of
Ankh-Morpork. Between the Most Ancient and Honourable Guild of Assessors, a
clerk who has kept the true books for fifty-three years without being asked,
ten thousand years of goblin accountancy, and certain grey gentlemen who feel
that being alive is an unlicensed irregularity, the city is about to discover
what is actually owed.

**Read it here: <https://rodalpho.github.io/raising-taxes/>**

Nine chapters and an epilogue, ~18,000 words.

## The edition

The site is a hand-built static "aged folio" edition: parchment and candlelit
themes, per-chapter engraved ornaments, drop caps, Going Postal–style chapter
arguments, and clickable footnotes that open as pop-up cards (and degrade
gracefully to endnotes).

- Source text: `raising-taxes-chapter-*.md`
- Generator: `python3 build.py` (stdlib only) → renders `docs/`
- Styles & behaviour: `assets/style.css`, `assets/book.js`
- Served by GitHub Pages from `docs/` on `main`
